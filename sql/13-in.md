# 13 - In

Pega linhas que batem com QUALQUER valor da lista. Mais limpo que vários `OR`:
```sql
SELECT titulo, genero FROM jogos WHERE genero IN ('FPS', 'RPG');
/*
╭───────────────┬────────╮
│ titulo        │ genero │
╞═══════════════╪════════╡
│ Valorant      │ FPS    │
│ CS2           │ FPS    │
│ The Witcher 3 │ RPG    │
│ Elden Ring    │ RPG    │
╰───────────────┴────────╯
*/
```
